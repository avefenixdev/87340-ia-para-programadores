## Usar OpenUI dentro de la compu

### Instalar Docker

<https://www.docker.com/>

### Activar la capacidad de virtualización
Si al ejecutar por primera vez Docker les va a decir que activen la virtualización si aún no lo tienen activa

```buscar-google
Como activar la virtualizacion de hardware en mi pc
``` 

**Nota**: Puede que les pida reiniciar

### Instalando OpenUI

<https://openwebui.com/>

<https://github.com/open-webui/open-webui>

> Terminal de Windows

```sh
docker run -d -p 3000:8080 --add-host=host.docker.internal:host-gateway -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:main
```

Se puede usar el agente de tareas que nos permite automatizar ciertas tareas como por ejemplo el calculo de totales dentro de una planilla de calculo de Google Sheet.

## Perplexity
<https://www.perplexity.ai/>

* Busca en la web en tiempo real
* Resume y explica la informacion encontrada
* Cita fuentes confiables (ideal para investigar)
* Permite hacer preguntas de seguimientos, como si fueran charlas con un asistente

## Google NotebookLM
Es un cuarderno de investigación asistido. Nosotros cargamos las fuentes que queremos que analice la IA y responde con la información que le pasamos.

* Estudiar: resúmenes fieles, mapas conceptuales, preguntas tipo examen.
* Docencia: guías, explicaciones graduadas, ejemplos sacados del texto.
* Investigación: encontrar relaciones, contradicciones y citas exactas.
* Trabajo serio: documentación, specs, papers, normativas.

<https://notebooklm.google/>
